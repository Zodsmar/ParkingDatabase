Parking and Date Database Search Engine

To use this software please input your License plate to start. Once you search the database for your license plate you can then select other methods of searching to narrow down the search. If you input a wrong search query, use the reset button to start over. (Ultimatly the final product will restrict the user to input a full license plate. For this assignment we left it open to only 1 character needed, to make searching easy.)

Once an item has been filter down, it has been deleted from the table. If you would like to go back you must restart. Also to search by date or time you must select one of the radio buttons then perform the search after a plate number has already been searched!
